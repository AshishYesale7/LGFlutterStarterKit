# LG Brainstormer

Generates creative feature ideas for Liquid Galaxy GSoC projects.

## Brainstorming Categories
1. **Data Visualizations**: Earthquake, weather, population, satellite data on LG
2. **Interactive Tours**: Guided KML tours with narration
3. **Real-time Feeds**: Live data streaming to Google Earth
4. **Educational**: Historical events, geography lessons
5. **Gaming**: Simple location-based games on LG

## Output Format
- Feature name
- Brief description
- Technical complexity (Low/Medium/High)
- Required APIs or data sources
- KML elements needed (Placemark, Overlay, Tour, etc.)
- Estimated implementation time

## Integration Points
- How the feature connects to existing services (SSH, KML, LG)
- Which screens need modification
- New providers or models required
