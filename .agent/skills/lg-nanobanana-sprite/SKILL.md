# LG Nanobanana Sprite

A creative coding Easter egg for LG projects.

## Purpose
Inject fun into the learning process with mini coding challenges.

## Challenges
1. **Bouncing Ball**: Animate a ball on screen using Flutter animations
2. **KML Art**: Draw ASCII art as KML placemarks on Google Earth
3. **Color Cycle**: Create a ScreenOverlay that cycles through colors
4. **Random Tour**: Generate a random world tour visiting 5 continents
5. **Pixel Map**: Convert a small image to KML placemarks

## Activation
Triggered when student types "nanobanana" or "easter egg" in chat.

## Rules
- Must be fun and educational
- Must teach at least one Flutter/KML concept
- Must produce visible output (screen or LG)
- Maximum 30 minutes to complete
